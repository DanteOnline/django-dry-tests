Install
-------

with pip
^^^^^^^^

.. code-block:: bash

   pip install django-dry-tests

from release page
^^^^^^^^^^^^^^^^^

Download source code from `GitHub Releases page <https://github.com/quillcraftsman/django-dry-tests/releases>`_

clone from GitHub
^^^^^^^^^^^^^^^^^

.. code-block:: bash

   git clone https://github.com/quillcraftsman/django-dry-tests.git
   make install
